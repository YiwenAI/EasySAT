Developer: SeedSolver Beijing 

Homepage: https://seedsolver.com

## How to Build

`make`


## How to use

`./EasySAT <cnf_file>`

The standard `.cnf` input format and the output format can be seen [here](http://www.satcompetition.org/2009/format-benchmarks2009.html)
